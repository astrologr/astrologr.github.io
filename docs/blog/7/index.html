5 σκληρές αλήθειες για το κάθε ζώδιο

  • Post by Efthy
  • Dec 19, 2021

Ποιο είναι το πιο γελοίο χαρακτηριστικό των Κριών και γιατί όλοι κράζουν τους Υδροχόους; Είναι ο Ταύρος όντως τόσο τεμπέλης, που μέχρι να σηκώσει το ένα πόδι έχει βρωμίσει το άλλο; Είναι οι Δίδυμοι ότι πιο διπρόσωπο θα συναντήσεις σε άνθρωπο; Αυτά και άλλα φλέγοντα ερωτήματα βρίσκουν επιτέλους την απάντησή τους μέσα από 5 fun facts για το κάθε ζώδιο.

Κριός

  1. Στις 10 λέξεις, οι 5 που θα πει είναι «Εγώ».
  2. Όταν ένας Κριός δηλώσει ότι θα κάνει κάτι, τότε θα το κάνει. Εκτός αν δεν πρέπει να το κάνει, οπότε θα το κάνει χωρίς να πει τίποτα.
  3. Και αν αποφασίσεις να του τη πεις που το έκανε ενώ δε θα έπρεπε να το κάνει, καλό θα ήταν να έχεις ψάξεις εκ των προτέρων ποιο νοσοκομείο εφημερεύει.
  4. Η ώρα που χρειάζεται ένας Κριός για να αποφασίσει αν θα αλλάξει ήπειρο είναι ίδια με αυτή που χρειάζεται για να αποφασίσει τι θα φάει σήμερα. Ίσως για το φαΐ να το σκεφτεί πιο πολύ.
  5. Ετοιμάζεσαι να αποκτήσεις παιδί με έναν Κριό; Μπράβο γλυκούλα! Τώρα θα νταντεύεις 2.

Ταύρος

  1. Είναι ότι πιο βαρεμένο, νωθρό και αργόσυρτο έχεις δει. Η εβδομαδιαία ανάγκη του σε φυσική άσκηση καλύπτεται με το να σηκωθεί από το καναπέ για να συρθεί μέχρι το κρεβάτι.
  2. Ο Ταύρος αγαπάει τη φύση! Είναι το μέρος όπου μπορεί να ηρεμήσει κ να μη μιλάει με κανέναν κάνοντας τίποτα.
  3. Έχει ανεξάντλητη υπομονή. Ο Βούδας μπροστά σε ένα Ταύρο φαντάζει σαν μια κατσαρίδα στα πρόθυρα νευρικής κρίσης.
  4. Του αρέσουν τα απλά πράγματα στη ζωή. Δηλαδή μαμ, κακά και νάνι.
  5. Θα κάνει τα πάντα για να σώσει τη σχέση του. Αν χωρίσει σημαίνει ότι πρέπει να ψάξει νέο σύντροφο. Και στη τελική, ποιος έχει χρόνο για αυτά, ιδιαίτερα εν καιρώ πανδημίας;

Δίδυμος

  1. Τους κατηγορούν ότι βαριούνται εύκολα. Στη πραγματικότητα ακόμα και ένα δευτερόλεπτο απραξίας μπορεί να τους κάνει να φερθούν σαν αυτόν τον υστερικό πρώην σου.
  2. Ξέρουν πως να προσαρμοστούν σε όλες τις καταστάσεις. Έχεις δει Δίδυμο να είναι παντρεμένος και ταυτόχρονα να το παίζει single;
  3. Είναι καλοί στο να κάνουν πολλά πράγματα ταυτόχρονα, αλλά ακόμα καλύτεροι στο να τα αφήνουν όλα μισά.
  4. Τους αρέσει να μαθαίνουν τα πάντα. Από το πρόσφατο νομοσχέδιο για τα εργασιακά μέχρι το γιατί η Κιμ Καρντάσιαν απέτυχε στις εξετάσεις για τη Νομική σχολή.
  5. Σταματήστε να κατηγορείτε τους Δίδυμους ότι είναι διπρόσωποι. Έχουν παραπάνω από 2.

Καρκίνος

  1. Δεν πετάνε ποτέ πράγματα συναισθηματικής αξίας. Θα τα κρατήσει όλα, ακόμα και τη τσίχλα που μασούσε ο πρώην του τη στιγμή που του έλεγε ότι θέλει να χωρίσουν.
  2. Θα ανανεώσει τη ντουλάπα του μια φορά στα 15 χρόνια.
  3. Αγαπάει οτιδήποτε vintage. Θυμάσαι αυτό το κακόγουστο τραπεζάκι του ‘50 που είχε η θεία Ζηνοβία στο χωριό; Σίγουρα πλέον ζει στο σαλόνι ενός Καρκίνου.
  4. Όταν νιώθει πεσμένος το ρίχνει στο φαΐ. Το φαΐ είναι αγάπη.
  5. Του αρέσει να αποκτάει άχρηστες ικανότητες. Μπορεί να ανοίξει μια μπύρα με τα δόντια του ή να αλλάξει τη πάνα ενός βρέφους και ας μην έχει ακόμα παιδιά.

Λέων

  1. Είναι απίστευτα γενναιόδωροι και τους αρέσει να αγοράζουν δώρα για τους φίλους τους. Το πρόβλημα ξεκινάει όταν οι φίλοι δεν ανταποδίδουν το δώρο.
  2. Τους αρέσει τόσο πολύ το θέατρο, που παίζουν ρόλους και στη πραγματική τους ζωή.
  3. Ένας Λέων είναι τόσο μετριόφρων όσο ενεργητικός είναι και ένας Ταύρος.
  4. Τους αρέσει να είναι το επίκεντρο της προσοχής. Ακόμα και αν αυτό σημαίνει ότι για να το καταφέρουν πρέπει να ντύνονται σα το Λάκη Γαβαλά.
  5. Αν ένας Λέων δεν είναι η νούμερο 1 προτεραιότητά σου, τότε στο μυαλό του κάνεις κάτι λάθος.

Παρθένος

  1. Για έναν Παρθένο το ποτήρι δεν είναι ποτέ μισοάδειο ή μισογεμάτο. Είναι πάντα βρώμικο.
  2. Είναι οι καλύτερες νταντάδες. Το έχουν εξασκήσει σε όλες τους τις σχέσεις.
  3. Δεν μπορούν να κοιμηθούν αν οι παντόφλες τους δεν είναι στη σωστή θέση. Οποιαδήποτε και αν είναι αυτή.
  4. Αν χωρίσεις με ένα Παρθένο δε θα σκάσει κιόλας. Ξέρει ότι σύντομα θα γυρίσεις παρακαλώντας τον να τακτοποιήσει το χάος της ζωής σου.
  5. Δεν πατάνε ποτέ σε ρωγμές στο πεζοδρόμιο. Κάτι τέτοιο θα διατάρασσε τη παγκόσμια ισορροπία.

Ζυγός

  1. Τους αρέσει να αναζητούν την ισορροπία περισσότερο από το να τη βρίσκουν.
  2. Είναι ένα πολύ επικοινωνιακό ζώδιο. Μην πάρεις ποτέ τηλέφωνο έναν Ζυγό, εκτός και αν έχεις κανένα 3ωρο για πέταμα.
  3. Ο Ζυγός είναι συνέχεια στην πένα. Τόσο πολύ που αναρωτιέσαι αν όταν κοιμάται δεν ακουμπάει στο μαξιλάρι για να μην χαλάσει το μαλλί.
  4. Αν ο Ζυγός σου κάνει ένα κομπλιμέντο, μην το πάρεις και πολύ πάνω σου. Το πιο πιθανό είναι ότι το κάνει από υποχρέωση.
  5. Αν θες να καταστρέψεις μια βραδινή έξοδο για φαΐ ενός Ζυγού, στο δίλλημά του «Κινέζικο ή σούσι;» απάντησε «Μήπως Ταϋλανδέζικο;»

Σκορπιός

  1. Του αρέσει να διαβάζει πίσω από τις γραμμές. Φήμες λένε ότι ο πρώτος συνωμοσιολόγος ήταν Σκορπιός.
  2. Ίσως ήταν και ο πρώτος χάκερ.
  3. Λίγοι άνθρωποι θα σε αγαπήσουν τόσο έντονα όσο ένας Σκορπιός. Τόσο έντονα που ίσως καταλήξει σε καταπίεση.
  4. Το χειρότερο πράγμα που μπορείς να κάνεις σε έναν Σκορπιό είναι να του λες συνέχεια την αλήθεια. Είναι σαν να του αφαιρείς τον σκοπό της ζωής του.
  5. Έχουν τη φήμη ότι είναι φιλύποπτοι. Δε θα σου δώσουν τον αριθμό τους εκτός και αν σε ξέρουν για χρόνια.

Τοξότης

  1. Αν θέλεις να φέρεις έναν Τοξότη στα όρια του, πες του τι δεν μπορεί να κάνει. Και μετά πάρε ποπ-κορν και απόλαυσε τον να προσπαθεί να το κάνει.
  2. Μπορεί να το παίζουν αγνωστικιστές αλλά με τη πρώτη δυσκολία αρχίζουν τις προσευχές και τις μετάνοιες.
  3. Τους αρέσει να επικεντρώνονται στο μέλλον. Ακόμα και αν αυτό το μέλλον είναι κάτι τόσο ασήμαντο όσο το τι θα κάνουν το σαββατοκύριακο.
  4. Είναι το πιο φιλοσοφικό ζώδιο του ζωδιακού κύκλου. Με την ίδια ευκολία που θα σου αναλύσει τις επιπτώσεις του Διαφωτισμού στην Ευρώπη, θα σου αναλύσει και τους κοινωνικούς λόγους που βοήθησαν στη επιτυχία προσωπικοτήτων όπως η Κιμ Καρντάσιαν και η Πάρις Χίλτον.
  5. Τους αρέσει η περιπέτεια. Το «Πάμε για ένα χαλαρό ποτό» του Τοξότη, συνήθως καταλήγει σε τραγωδία.

Αιγόκερως

  1. Τείνει να μπαίνει σε σοβαρή σχέση κάπως αργά στη ζωή του: «Δε φταίω εγώ που είμαι ανεξάρτητος και αυτάρκης»
  2. Αν θέλεις να προσβάλεις έναν Αιγόκερω πες του «Πολύ ωραίο το μπλουζάκι σου. Θέλω και εγώ! Είναι τόσο φθηνό όσο φαίνεται;»
  3. Όταν ο Αιγόκερως θέλει κάτι πολύ, το αναλύει μέχρι αηδίας και στο τέλος καταλήγει στο ότι το σιχαίνεται.
  4. Αν θέλεις να χωρίσεις με έναν Αιγόκερω χωρίς να τον πληγώσεις, πες του ότι έκλεισες ταξίδι για 2 για να εξερευνήσετε την Παταγονία. Το πιο πιθανό είναι ότι θα εξαφανιστεί.
  5. Όλοι έχουμε ένα φίλο που περιμένει ακόμα έναν Αιγόκερω να κάνει τη πρώτη κίνηση.

Υδροχόος

  1. Η πρώτη μάνα που είπε στο παιδί της «Φάε όλο το φαγητό σου. Τα παιδάκια στην Αφρική πεινάνε» ήταν Υδροχόος και εννοούσε κάθε λέξη.
  2. Οι Υδροχόοι είναι εκπληκτικοί φίλοι. Θα σε συμβουλεύαμε να μείνει στο φιλικό και να μην πάει παραπέρα.
  3. Δεν έχουν αναστολές. Είναι ελεύθερες ψυχές. Τόσο ελεύθερες που η συμπεριφορά ενός Υδροχόου έξω μπορεί να κάνει ρεζίλι όλη τη παρέα.
  4. Στο μυαλό του Υδροχόου οι κανόνες είναι για να σπάνε. Αυτή η σκέψη κρατάει μέχρι να έρθουν αντιμέτωποι με τις συνέπειες.
  5. Όλοι ξέρουμε έναν Υδροχόο που ακόμα μας παίρνει τηλέφωνο για να βγούμε εκείνο το ραντεβού που λέγαμε.

Ιχθύς

  1. Αν δεις έναν Ιχθύ να μιλάει μόνος του, μην τον παρεξηγήσεις. Προφανώς και τα λέει με το φανταστικό του φίλο.
  2. Θυμάσαι το Nemo? Τα ψάρια είναι επιρρεπή στον αποπροσανατολισμό.
  3. Αν δεις κάποιον να είναι ντυμένος λες και ξεπετάχτηκε από το ’50, οι πιθανότητες είναι ότι είναι Ιχθύς.
  4. Το ήξερες ότι τα ψάρια κοιμούνται με τα μάτια ανοιχτά. Αυτό δεν είναι σπόντα για το πόσο κουτσομπόληδες είναι οι Ιχθύες.
  5. Μπορούν να θυμηθούν τι έκαναν τέτοια μέρα 15 χρόνια πριν, αλλά αδυνατούν να σου πουν τι έφαγαν χθες για βραδινό.
